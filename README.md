# ShortcutCreator
A program to let you create shortcuts easily

## Notes:
- Shortcut recording works by reading the keys in a certain way within a 5 second period. Results are displayed for 3 seconds after.
- Some keyboard keys may not work when generating a shortcut (IWshShortcut has restrictions)

Examples:
- (Key) => Ctrl + Alt + (Key)
- (Key) + (Shift) => Ctrl + Shift + (Key)
- (Shift) + (Key) => Ctrl + Shift + (Key)
- (Shift) + (Key) + (Alt) => Ctrl + Alt + (Key)
> Ctrl will always be first!
